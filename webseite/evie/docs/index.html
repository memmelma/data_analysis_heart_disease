Risikofaktoren für Herzkrankheiten erkennen

Ein weit verbreiteter Anwendungsbereich von Machine Learning ist die medizinische Diganostik, die durch ihre Interaktion mit künstlicher Intelligenz tiefgründigere Zusammenhänge zwischen Krankheiten und ihren Ursachen aufdecken kann. Birgt die maschinell gestützte Diagnose selbst einige Risiken, profitieren Ärzte dennoch durch die Unterstützung eines repräsentativen, trainierten Modells, da Erfahrungswerte aus Millionen von Patientendaten zu einer Diagnose heranzgezogen werden können.

Was wollten wir erreichen?

Ziel ist mithilfe eines geeigneten Modells die Praesenz einer Herzkrankheit moeglichst genau vorherzusagen. Das Risiko einer Falschdiagnose soll hierbei hinsichtlich gravierender medizinischer Konsequenzen minimiert werden. Bezogen auf die Machine Learning Problematik kann dieses Problem als Binary Classification beschrieben werden.

Was haben wir erreicht?

Durch das Trainieren verschiedener Modelle und Optimierung der zugehoerigen Stellschrauben (Hyperparameter), konnten mehrere Loesungsansaetze evaluiert werden. Durch Einsatz des K-Nearest-Neighbor Algorithmus wurde eine Accuracy von ueber 0.9 erreicht. Dies bedeutet, dass auf Basis der eingegebenen Daten eine Herzkrankheit zu 90% festgestellt werden kann.
